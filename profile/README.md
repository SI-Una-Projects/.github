# SI-Una-Projects

🚀 Organização criada para reunir projetos acadêmicos, atividades práticas e trabalhos desenvolvidos durante o curso de **Sistemas de Informação da UNA**.

Aqui estão centralizados projetos desenvolvidos em equipe e individualmente, utilizando tecnologias modernas voltadas para desenvolvimento web, back-end, banco de dados e boas práticas de engenharia de software.

---

## 🎯 Objetivo

A organização tem como objetivo:

- Centralizar os projetos acadêmicos
- Facilitar colaboração entre integrantes
- Versionar códigos utilizando Git e GitHub
- Documentar a evolução dos projetos desenvolvidos durante a graduação
- Aplicar práticas do mercado em projetos reais

---

## 💻 Tecnologias Utilizadas

Algumas tecnologias utilizadas nos projetos:

- Java
- Spring Boot
- React
- TypeScript
- MySQL
- HTML / CSS
- Git & GitHub
- APIs REST
- Scrum e metodologias ágeis
